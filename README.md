IBM HR Analytics - Employee Attrition_Analysis
##Project Overview
End-to-end HHR analytics project analyzing why employees leave,
using Python for data Cleaning and Power BI for Visualization.
## Tools & Skills Used
-Python (Pandas) - data cleaning, null removal, feature engineering.
-Power BI - 5-page interactive dashboard
-Dataset: IBM HR Analytics (Kaggle)
## Data Cleaning Steps (Python/Pandas)
-Removed null values and duplicate rows.
-Added new engineered columns (Age_Group,etc.)
-Exported clean data as HR_cleaned.csv
## Dashboard Pages
1. Executive Summary
2. Attrition Analysis
3. Employees Profile
4. Carrer Health
5. Key Insights
## Key Findings
- Overall Attrition Rate: 16.1%
- Total Employees Who Left: 237
- Avg Monthly Income of Leavers: 4.79k
- Overtime is #1 driver - 30.5% vs 10.4% attrition
- Early Career employees have 36% attrition rate
- Compensation gap: leavers earned 4.8k vs 6.8k (29% less)
